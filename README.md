# Sahey Mobile App

## Run the app on your phone

1. **Download Expo Go** on your phone from the [App Store](https://apps.apple.com/app/expo-go/id982107779) (iOS) or [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent) (Android).

2. **Start the dev server with tunnel mode** (so your phone can connect from any network):
   ```bash
   npx expo start --tunnel --go
   ```

3. **Scan the QR code** shown in the terminal using your phone's camera app. Tap the notification or link that appears to open the project in Expo Go.

4. The project will open in Expo Go and the app will load on your phone.

Make sure you have Node.js and npm installed. Run `npm install` in the project folder if you haven't already.
